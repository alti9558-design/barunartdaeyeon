
## 섹션 수정
유치부 소묘 섹션을 삭제하고 초등부 저학년 섹션으로 변경했습니다. 사진은 1장만 사용하도록 정리했습니다.

## 추가 수정
수업요일과 상담 안내 문구 추가: 화~토 수업, 평일 1시~10시, 토요일 9:30~2시, 추천 요일 수/금, 미리 연락 시 상담 가능.

## 문구 수정
대표 문구를 “분석하고 훈련하는”에서 “꿈을 이루는 바른 가르침”으로 변경했습니다.

## 버튼 수정
메인 화면의 입시분석보기, 재현작보기 버튼을 제거했습니다.

## 안전 재빌드
앞부분 섹션이 사라지는 문제를 방지하기 위해 정상 버전에서 다시 누적 수정했습니다. 성인취미 사진/문구, 전국 예중예고 FAQ, 주전자 공개시범작, 체험 수업 상담 FAQ, 총 112명 합격 문구를 반영했습니다.

## 수정
FAQ에서 수업요일/수업시간 중복 질문을 제거했습니다. 메인 화면 이미지는 다시 총 112명 합격 이미지로 복구했습니다.

## 수정
프로그램 카드의 초등부 저학년 항목을 제거했습니다. “주전자 1시간 공개시범작” 앞의 “그림 제목” 문구를 제거했습니다.

## 이미지 누락 방지
홈페이지에서 참조하는 모든 이미지 파일을 assets 폴더에 포함했습니다. GitHub 업로드 시 index.html과 assets 폴더 전체를 함께 올려야 합니다.

## 학교 자료 섹션 수정
부산권만 상담하는 느낌을 줄이도록 문구를 전국 예중·예고 준비로 변경했습니다. 부산예중/부산예고, 브니엘예중/브니엘예고가 붙어 보이도록 순서를 재배치하고, 빈칸에는 추천 로고 이미지를 추가했습니다.

## 최종 다듬기
입시분석의 버튼형 문구(모집 인원 확인 등)를 제거했습니다. 초등 저학년 문구의 “아이의 속도”를 “아이의 진도”로 변경했습니다. 수업시간 안내의 추천 요일 문구를 “상담 후 선택 / 시간과 요일 조율”로 정리했습니다.

## PC 메인 화면 수정
PC에서 메인 이미지가 더 크게 보이도록 조정하고, 메인 문구를 보기 좋게 줄바꿈했습니다. 모바일은 기존 1열 반응형을 유지했습니다.

## 최종 수정
수업시간 섹션의 상담 후 선택/시간과 요일 조율 행을 제거했습니다. 체험 수업 FAQ를 체험 수업 상담 문구로 고정했습니다. PC 메인 이미지 크기를 줄이고 텍스트가 가려지지 않도록 hero 영역을 재조정했습니다.

## 문구 수정
홈페이지 전체에서 “부산권” 표현을 제거하고 전국 예중·예고 준비 문구로 정리했습니다.

## 수업사진 추가
학원 수업사진 5장을 class-photo01~05로 assets 폴더에 추가하고, “학원 수업사진” 섹션을 새로 배치했습니다.

## 수업사진 수정
오일파스텔 표현 수업 사진을 제거하고, 남은 수업사진 4장을 균형형 갤러리로 재배치했습니다.

## 문구 수정
초등 저학년 섹션의 “한 장의 사진으로 간결하게 보여줍니다” 문구를 빼고 맞춤수업 강조 문구로 변경했습니다. FAQ 질문에서 “전국” 표현을 빼고 “예중·예고도 준비하나요?”로 정리했습니다.

## FAQ 수정
“예중·예고도 준비하나요?” 질문을 “목표 학교별 준비가 가능한가요?”로 변경했습니다.

## 색상 수정
입시요강 분석 섹션의 설명 문구가 회색으로 보이던 문제를 수정하여 흰색으로 표시되도록 했습니다.

## FAQ 수정
체험 수업 FAQ를 “체험 수업이 가능한가요? / 네, 체험 수업이 1회 가능합니다...” 문구로 다시 고정했습니다.

## 수업사진 보이도록 재배치
수업사진 섹션이 보이지 않는 문제를 해결하기 위해 “학원 수업사진” 섹션을 메인 핵심 요약 바로 아래로 올리고, 이미지 4장을 다시 연결했습니다.

## sitemap 보강
sitemap.xml 파일을 추가했습니다. GitHub에는 index.html, assets 폴더, robots.txt, sitemap.xml을 함께 올려주세요.
